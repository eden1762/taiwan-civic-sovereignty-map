本網站只整理公開登記資料與主辦方公開公告，讓台灣公民、主權派人士、中立人士、守法人士與年輕人都能快速理解：誰在辦、在哪裡、訴求是什麼、怎麼查證。

你可以看到主辦方、協辦方、訴求、公開地址、活動時間、參加資格與活動連結。資訊以官方公告為準，請出發前再次確認。

Data cards

公開資料卡

點選卡片即可在地圖定位。活動類資料會用範圍圈提醒「可能活動區域」，不是追蹤私人行程。

資料來源與守法界線

本網站預設採用政府登記、政黨官網、組織官網與公開活動報名頁。只放「公開地址」與「公開活動」，不放私人住址、不公開聚會、不做個人追蹤。

資訊可能變動

政黨、協會、基金會地址與活動時間都可能更新；每張資料卡都有「更新狀態」與「官方連結」。出門前請再查一次，才不會白跑，no wasted trip。

Easy maintenance

維護方式：只改 data.js

這個網站沒有 React、沒有 build、沒有 dist。你要新增或刪除資料，只要打開 data.js，修改 organizations 或 events 陣列即可。

  1. 新增組織：複製一筆 organization，改名稱、地址、座標、來源連結。
  2. 新增活動：複製一筆 event，改日期、位置、資格、報名網址。
  3. 部署：把檔案推到 GitHub 連 Vercel，或直接 Vercel 匯入此資料夾。
